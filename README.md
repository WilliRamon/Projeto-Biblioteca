# Sistema de Controle para Biblioteca
 Utilizando conceitos de Orientação a Objeto, o sistema possibilita que o usuario consiga:
  - Consultar o Estoque;
  - Consultar Cadastros dos Clientes;
  - Cadastrar Exemplar;
  - Cadastrar Cliente;
  - Emprestar Exemplar;
  - Devolver Exemplar;
  - Excluir Exemplar;
  - Excluir Cadastro de Cliente;
  - Consular Pendências.
